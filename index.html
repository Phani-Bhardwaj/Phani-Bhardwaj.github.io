01 / SIGNATURE PROJECT

A five-node pipeline, not a chatbot wrapper

Most "AI agent" portfolio projects are RAG demos over a PDF. This one catches real security misconfigurations in Terraform & Kubernetes before they reach production — combining deterministic static analysis with LLM reasoning, grounded by retrieval, with every fix traceable to a source.

Generates fixes with citations back to the knowledge base

STEP 5
📋

report_agent

Compiles findings + evidence into one markdown report — no LLM call

WHY CHECKOV + LLM, NOT LLM-ONLY

Checkov catches known misconfiguration patterns deterministically — zero hallucination risk. The LLM pass catches what static rules can't: setting combinations that add up to a privilege-escalation path even when no single rule fires.

WHY RAG, NOT DIRECT LLM CALLS

Ask an LLM "why is this insecure?" with no retrieval and it answers from parametric memory — which drifts and can't be audited. Every remediation here cites a specific knowledge-base entry, so it's checkable, not just trusted.
